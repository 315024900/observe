

<h2 align="center">mini Vue</h2>
 一个精简版的vue，帮助你理解vue的源码逻辑。
 已经实现的功能：
 - 事件绑定
 - template模板功能编译。
 - html标签属性绑定。
 - vue的{{}}}参数绑定。
 - 精简的Observer系统，变量改变，自动更新html。



# mini Vue  
 一个精简版的vue，帮助你理解vue的源码逻辑。 
 
## 已经实现的功能：  
1. 事件绑定  
2. template模板功能编译。  
3. html标签属性绑定。  
4. vue的{{}}}参数绑定。  
5. 精简的Observer系统，变量改变，自动更新html。
